# Caleb Harmon - Personal Website Template

A modern, responsive personal website template for a data engineer, built with vanilla HTML, CSS, and JavaScript.

## Features

- **Hero Section**: Full-screen intro with typing animation and floating shapes
- **Responsive Design**: Works perfectly on desktop and mobile
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Scroll Animations**: Elements animate in as you scroll
- **Glassmorphism Effects**: Modern glass-like sections with blur
- **Interactive Navigation**: Active link highlighting and mobile menu
- **Progress Bar**: Visual scroll progress indicator
- **Contact Section**: Social links with icons
- **Custom Fonts**: Google Fonts for professional typography
- **Font Awesome Icons**: Icons throughout the site
- **Scroll to Top**: Convenient navigation button

## Technologies Used

- HTML5
- CSS3 (with Flexbox, Grid, and Animations)
- Vanilla JavaScript
- Google Fonts
- Font Awesome

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser or serve with a local server
3. Customize the content to match your information

## Customization

- **Personal Information**: Update name, bio, and contact details in `index.html`
- **Skills**: Modify the skills section to reflect your expertise
- **Projects**: Replace placeholder projects with your actual work
- **Colors**: Adjust gradients and colors in `styles.css`
- **Animations**: Modify timing and effects in `script.js`

## File Structure

```
/
```

## Browser Support

This template works in all modern browsers that support CSS Grid, Flexbox, and ES6+ JavaScript features.
